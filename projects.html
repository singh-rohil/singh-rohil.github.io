LightEmAll – Wire Puzzle Game

Interactive puzzle game where players rotate wire tiles to connect all cells to a power source. Generates solvable puzzles using Kruskal's Minimum Spanning Tree algorithm with randomized edge weights, ensuring every puzzle has a valid solution.

Key Features

  • Kruskal's MST algorithm for guaranteed-solvable puzzle generation
  • Breadth-First Search (BFS) for power propagation through connected tiles
  • Horizontal/vertical bias modes for varied puzzle styles
  • Moveable power station with keyboard controls
  • Score tracking with clicks, timer, and best score
